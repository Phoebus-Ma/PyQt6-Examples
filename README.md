# Platform

Python 3 on Windows/Linux/MacOS.


# Enviorment

Install pyqt6:

```bash
$ python -m pip install --upgrade pip
$ pip install pyqt6
```

Or, you can install pyside6:

```bash
$ python -m pip install --upgrade pip
$ pip install pyside6
```

PyQt6 and PySide6 are like twins, here uses PyQt6.
